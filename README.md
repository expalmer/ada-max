# Ada Max

Criando a aplicação usando o Vite.

```bash
npm create vite@latest ada-max -- --template react-ts
```

Colocando React Router

```bash
npm i react-router-dom
```

## 1. Iniciando a aplicação

```bash
npm run dev
```
